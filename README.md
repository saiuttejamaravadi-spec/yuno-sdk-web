This repository is based on the official Yuno demo application.

## What was done
- Cloned and configured the official Yuno demo repository
- Installed Node.js dependencies
- Configured Yuno Testing Gateway in Dashboard
- Executed an embedded checkout using SDK FULL
- Completed a successful credit card test payment
- Verified transaction in Yuno Dashboard

## Flow Overview
1. Backend creates a payment session using Yuno API
2. Frontend initializes Yuno SDK FULL
3. Embedded credit card checkout is rendered
4. Test card payment is processed
5. Transaction is reflected in the Dashboard

## Notes
- This demo focuses on validating the payment flow
- Frontend and backend are provided by Yuno demo app
- No UI customization was done as it was not required for this assessment
